# 서빙고 PROJECT 

개요 :
 [ 갑자기 웬 서빙고 프로젝트? ]
 - 뒤 늦게 코딩에 흥미를 갖게 된 이 모씨(당시 한국식 나이 25세)는 코딩 연습 겸
   재밌고 신박한 아이디어 없을까 하다가 일단 처음엔 내 이름을 딴 무언가가 좋겠다
   생각이 들어 하나하나 이... 흠.. 준... 흠.. 서.... 흠.. 하는 고민을 하게 된다.
   그러다 문득(정말 문득) 예전에 내가 신기하게 본 1호선 서빙고 역이 생각났다!
   아하! 그래! 내 이름을 딴 빙고 판이나 만들어보자! 해서 만들어 진 게 서빙고 프로젝트다(...)

로직 :
 [ 컴퓨터와 함께하는 재미난 빙고 ]
 - 이 서빙고 프로그램은 60개의 단어를 제공하여, 사용자가 원하는 단어를 4X4 칸에 넣고,
   컴퓨터가 60개의 단어 중 랜덤으로 하나씩 말하면서 3빙고(가로, 세로, 대각선)을 맞추면
   되는 프로그램이다. 계속 눌러서 맞추면 재미가 없으니, 컴퓨터와 플레이어 간 승패를 줄
   수 있는 조건이 하나 있다.

   버튼은 총 30번만 누를 수 있다. 30회 안에 3빙고가 맞춰진다면 플레이어 승리. 그렇지 못
   할 경우, 컴퓨터의 승리다. 
   
마치며 :
 [ 첫 프로젝트 이자 최고의 공부가 될 것. ]
 - 뭔가 부실하다 느껴진다면 정답이다. 갑자기 아이디어 번뜩이고(10분전), VS Code 켜서(5분 전)
   이 텍스트 글 쓰고(지금)있으니 말이다. 일단 틀은 이렇게 잡고, 점차 세부적으로 살을
   붙여가면서 만들어가려 한다. 참고로 나 코딩 본격적으로 배운 지 얼마 안됐다.
   일단 Python으로 해보고, 재밌다면 Java로도 구현해 볼 예정이다. 기한은 딱히 정해두 지
   않지만, 하나 짤 때마다 무엇을 배웠고 이뤘는 지 코멘트를 달 생각이다. 코멘트 방식은 딱히
   뭘 배운 적이 없어서 내 방식대로 코멘트를 남기겠다.

   
   그럼 가봅시다.
